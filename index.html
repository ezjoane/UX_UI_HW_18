My Ux Skills

USER RESEARCH

I usually love to use 1-1 interview, survey, and proto persona to capture users’ pain, needs, and behaviors.

USER EMPATHY

Empathy is an ability that UX designers need to have to fully understand users’ needs, goals, and motivations. The tools that I used to do the user empathy are affinity diagram, empathy map, user persona, user journey map, etc.

QUALITATIVE INTERVIEWS

1-1 interview is an example of qualitative interviews. I can quickly get detailed feedback from the test user and ask some follow-up questions to them.
